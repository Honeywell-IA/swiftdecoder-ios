This enumeration represents all available Activation responses.

Enumerator
FAILED_UNKNOWN 

Activation Failed. Please check network related issues with your environment and honeywell servers.

SUCCESS 

Activation was successful.

FAILED_LICENSE_PERPETUAL 

License perpetual.

FAILED_LICENSE_INACTIVE 

License inactive.

FAILED_BLANK_ID 

Entitlement or Activation ID is blank.

FAILED_BLANK_ARGUMENT 

URL is blank.

FAILED_NOT_CREATED 

Activation failed, License Manager not created. Create License Manager Object.

FAILED_INSUFFICIENT_MEMORY 

Activation failed, LM has insufficient memory.

FAILED_ALREADY_CREATED 

Activation failed, LM already created.

FAILED_FOLDER_PATH_ERROR 

Activation failed, LM cannot write to specified path.

FAILED_DEVICEID_ERROR 

Activation failed, LM device ID error.

FAILED_DEVICETYPE_ERROR 

Activation failed, LM device type error.

FAILED_CUSTOMERID_ERROR 

Activation failed, License Manager customer ID error. Invalid Entitlement/Activation id. Please check the entitlement id on the license manager portal and reach out to Honeywell support.

FAILED_DEACTIVATION_NOT_ALLOWED 

Deactivation is not allowed for your license type. Please reach out to Honeywell support to turn this feature on.

FAILED_CURL_FAILURE 

Activation failed, no valid activation id attached to the entitlement Id, please check the entitlement id on the license manager portal and reach out to Honeywell Support.

FLEXERA_FAILURE 

Activation failed, Communication with server was succesfull, but activation has failed for an unknown reason. Please reach out to Honeywell support.

FAILED_INVALID_KEY 

Invalid license key.

FAILED_ALL_FEATURES_EXPIRED 

All the features of the key were expired.

FAILED_CLOCK_WINDBACK_DETECTED 

System time has been altered.

FAILED_FLEXERA_COM 

Activation failed, Communication with server was successful, but it is not responding. Please reach out to Honeywell support.

FAILED_FLEXERA_NO_RESPONSE 

No response from server, please check the access to Internet.

FAILED_CURL_ERROR 

Curl returned an error while contacting the server.

FAILED_CURL_HOST_NOT_FOUND 

Couldn't resolve host name.

FAILED_NOT_ACTIVATED 

Deactivation failed, not activated.
