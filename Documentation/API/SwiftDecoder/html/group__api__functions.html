◆ LicenseManagerDestroy()

void LicenseManagerDestroy ( void  )

Destroys the currently created Manager.

Returns
nothing

Only 1 Manager is ever created no matter how many times the create functions are called.
This function will free any memory allocated as part of a Create call, but will not destroy any files and storage created. If there is no Manager, this function will do nothing, so it is safe to call before a Create call.

◆ LicenseManagerGetLastErrorString()

char * LicenseManagerGetLastErrorString ( )

Returns string description of the last error.

Returns
String description of last error.

Most of the strings returned from this function come straight from the third party error strings. A Curl error for example simply returns the error code from Curl itself, added to a base number to identify it from any other third party error and this string is taken directly from the Curl Error String.
